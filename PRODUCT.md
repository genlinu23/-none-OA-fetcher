# Product

## Register

product

## Users

Researchers using a local Windows machine to collect academic PDFs from DOI lists or from a small research-topic search flow.

## Product Purpose

This app turns DOI input into organized local PDF download runs. The primary task is fast, controlled DOI download with visible progress, output paths, and result review.

## Brand Personality

Calm, scholarly, precise.

## Anti-references

Do not make the interface feel like a marketing landing page, school anniversary poster, decorative dashboard, or animation-heavy SaaS demo. Avoid large hero sections, full-screen splash screens, repeated stage markers, and showing every technical option at once.

## Design Principles

- Put DOI download controls in the first viewport.
- Show only the next useful action by default.
- Keep research search and title review available, but secondary.
- Hide advanced runtime settings until needed.
- Use screenshots to verify desktop and mobile layout before calling the UI done.

## Accessibility & Inclusion

Use readable contrast, stable focus states, reduced motion safety, and mobile layouts where the primary action appears before secondary diagnostics.
